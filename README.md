## 1. Provisional Patent – Footwear Securing Device

### Overview

Designed and filed a U.S. provisional patent for a footwear securing method and insole-integrated tension clamp system intended to stabilize tongue movement and improve fit integrity.

The invention includes:

- Insole-integrated strap system  
- Dual tension clamp mechanism  
- Multi-tooth clamping plates  
- Locking and ladder-lock adjustment system  
- Adhesive base stabilization  

This project required mechanical ideation, technical drafting, prototyping coordination, and formal patent documentation.
